# WorkSmartNotHard

WorkSmartNotHard είναι μια εφαρμογή iOS που βοηθά στη διαχείριση στόχων, καθημερινών εργασιών και παρακολούθησης προόδου. 

## Χαρακτηριστικά
- Κατηγορίες εργασιών και στόχων
- Καθημερινή καταγραφή δραστηριοτήτων
- Ιστορικό και στατιστικά
- Προσαρμογή θεμάτων και ρυθμίσεων

## Δομή έργου
- App: Κύριος κώδικας εφαρμογής
- Models: Μοντέλα δεδομένων
- Views: SwiftUI Views
- Resources: Πόροι εφαρμογής

## Εκκίνηση
1. Άνοιξε το project στο Xcode
2. Τρέξε το σε προσομοιωτή ή συσκευή

## Απαιτήσεις
- Xcode 15+
- iOS 17+
